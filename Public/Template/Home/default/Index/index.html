Note chrome插件

Note Chrome插件,名字太长了，我们就叫他“NC”吧， ···算了，不太好听。他是一个能够将你想要收藏的链接快速的放到Note中， 其中包括链接的地址，标题，等(为什么加了个“等”， 因为我感觉还有强的功能我没发现_(:з」∠)_)

Note 火狐插件

Note 火狐插件,“NF”这个还挺好听。 他的功能和chrom插件应该是相同， 为什么说是应该，因为还特么没开发···_(:з」∠)_ 详情见首页“模块开发进度”敬请期待···
